# Funções

Em programação, funções são reconhecidas por causar 
um reaproveitamento de código.

* rgb()
* hsl()
* url()
* calc()



<!-- exemplo

HTML:
<div class="box"></div>


CSS:
body {
  height: 100vh;
  margin: 0;
}
.box {
  height: calc(100% - 20%);
  width: 100%;
  background-image:
url(http://source.unsplash.com/random);
  background-repeat: no-repeat;
  background-position: center;
}
