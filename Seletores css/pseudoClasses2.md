## Ações do usuario

* :hover
* :focus

# Atributos

* :disabled
* :required

EXEMPLO:

```html
<a href="#">Clique aqui</a>
<input type="text" disabled required>
```


```css
a:hover {
    color: red;
}
input:focus {
    border-color: blue;
    outline: none;
}
input:disabled {
    background-color: green;
}
or
input:required {
    background-color: green;
}
```
